# إضافة حساب أدمن

هناك طريقتان لإضافة حساب أدمن جديد:

## الطريقة الأولى: من خلال واجهة المستخدم (مُوصى بها)

1. سجّل الدخول كأدمن في النظام
2. اذهب إلى صفحة **إدارة الموظفين** (`/admin/workers`)
3. اضغط على زر **"إضافة أدمن"** (الأحمر)
4. املأ البيانات المطلوبة:
   - الاسم الكامل
   - اسم المستخدم
   - كلمة المرور (4 أحرف على الأقل)
   - تأكيد كلمة المرور
   - رقم الهاتف (اختياري)
   - المنصب (افتراضي: "مدير")
5. اضغط على **"إنشاء حساب أدمن"**

## الطريقة الثانية: من خلال السكريبت

يمكنك استخدام السكريبت لإضافة حساب أدمن من سطر الأوامر:

```bash
npm run add-admin
```

أو مع قيم مخصصة:

```bash
ADMIN_USERNAME=admin2 ADMIN_PASSWORD=password123 ADMIN_NAME="مدير جديد" npm run add-admin
```

**ملاحظة:** السكريبت قد يحتاج إلى صلاحيات Firebase. إذا واجهت مشكلة في الصلاحيات، استخدم الطريقة الأولى.

## البيانات المطلوبة

- **الاسم الكامل**: اسم المدير الكامل
- **اسم المستخدم**: يجب أن يكون فريداً (غير مستخدم من قبل)
- **كلمة المرور**: 4 أحرف على الأقل
- **رقم الهاتف**: (اختياري)
- **المنصب**: (افتراضي: "مدير")

## الصلاحيات

حساب الأدمن يحصل تلقائياً على:
- ✅ صلاحيات كاملة للنظام
- ✅ الوصول إلى جميع الصفحات
- ✅ إدارة الموظفين
- ✅ إدارة المنتجات
- ✅ إدارة الطلبات
- ✅ التقارير والإحصائيات
- ✅ إدارة المخزن
- ✅ إدارة الطاولات والغرف

## الوصول المباشر

يمكنك الوصول مباشرة إلى صفحة إضافة الأدمن من:
- `/admin/add-admin`
